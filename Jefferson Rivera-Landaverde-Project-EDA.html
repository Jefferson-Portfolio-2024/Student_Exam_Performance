PART (D) Outliers

# corr matrix without removing some outliers
corr_studentPerformance <- cor(StudentPerformance, use = "pairwise.complete.obs")
corrplot(corr_studentPerformance, method = "square")

# Here we can see some outliers in our data. Lets remove some of the most extreme ones to see if they are influencial. 

# removing 5 of the most extreme outliers

# to select the outliers, I chose the Exam_scores furthest from the mean. so I noticed on the scatterplot that there was only one perfect score. I then got 4 more rows with Exam_scores above 97, which were the most extreme outliers in the dataset. 
Stud_no_out <- StudentPerformance[-c(4780, 95, 2426, 4298, 3580),]
cor_stud_no_out <- cor(Stud_no_out, use = "pairwise.complete.obs")
corrplot(cor_stud_no_out, method = "square")

# Removing the 5 most extreme outliers in this data set really did not change anything in the correlation plot. There were only slight increases in Exam_score ~ Attendance, and Exam_Score ~ Hours_studied, but not enough for the outliers to be considered significant. Therefore outlier influence should not be an issue in this data set.
